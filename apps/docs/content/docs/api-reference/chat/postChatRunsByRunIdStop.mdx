          Aborts the identified in-flight agent run server-side. The run must
          have been started by the same user/org identity on this instance;
          anything else — finished, unknown, another tenant's, or streaming on a
          different instance — returns 404. A stopped turn ends cleanly:
          completed steps stay persisted, no Resume affordance is offered, and
          the conversation accepts the next message immediately. Idempotent in
          effect — stopping an already-settled run is a 404 the client treats as
          'already finished'.
