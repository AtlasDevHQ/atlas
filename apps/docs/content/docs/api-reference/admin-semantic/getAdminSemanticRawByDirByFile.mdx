Serves raw YAML for an entity or metric (e.g. entities/users.yml). When an internal DB is configured and the caller has an active org, the DB row's `yaml_content` is canonical; the disk file under `semantic/` is a self-hosted fallback only. Pass `?connectionGroupId=<group>` to disambiguate when the same entity name exists in multiple environments.
