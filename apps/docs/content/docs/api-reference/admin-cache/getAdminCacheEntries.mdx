          Lists the caller's org's live cached entries as metadata rows (age,
          row count, connection, capped SQL preview) — expired entries never
          appear and the cached rows themselves are never returned. A platform
          admin may pass ?orgId= to inspect a specific org. `entries: null`
          means the listing is unavailable on the current cache backend; an
          empty list may also mean caching is disabled for the listed org.
