Globally bans a user across every workspace they belong to. Requires `platform_admin` role — workspace admins cannot perform a cross-tenant ban because the effect propagates past their workspace boundary. Workspace admins should use `DELETE /users/{id}/membership` to remove a member from their own workspace only (F-14, 1.2.3 phase 2).
