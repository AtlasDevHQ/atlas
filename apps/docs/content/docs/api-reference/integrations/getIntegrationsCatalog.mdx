      - content: >-
          Returns every enabled catalog entry visible to the requesting
          workspace. Rows above the workspace's plan tier are returned with
          `upsellOnly: true` so the UI can render a read-only upsell card. On
          SaaS deploys, `saas_eligible = false` rows are hidden entirely. The
          `installed` boolean reflects whether the workspace has a row in
          `workspace_plugins` for the catalog entry. Pass `?pillar=datasource`
          to list datasource-pillar catalog rows (with install status), or
          `?pillar=knowledge` to list Knowledge Base connector rows, instead of
          the default chat/integration listing.
