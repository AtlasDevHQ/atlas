Flip an archived connection's status back to `published` and cascade its semantic entities back to `published`. When the id is the reserved `__demo__` connection, built-in demo prompt collections matching the org's `demo_industry` setting are also restored. Returns 404 when the connection does not exist or is not currently archived.
