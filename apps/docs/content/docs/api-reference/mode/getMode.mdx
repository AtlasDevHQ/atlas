Returns the resolved mode for the current request, whether the user can toggle it, demo workspace state, and per-table draft counts. The frontend uses this to render banners, badges, the publish button, and the pending-changes summary. Non-admin users always receive `mode: 'published'` and `canToggle: false` regardless of the `atlas-mode` cookie or `X-Atlas-Mode` header.
